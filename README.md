### 프로젝트 소개

- 건국대학교 모바일 프로그래밍 개인 프로젝트.
- 식약처에서 관리하는 물품들 중 허위 / 과대 광고 여부가 있는 물품을 검색할 수 있고, 관련된 물품들에 대한 정보, 의견을 유저들 간 나눌 수 있는 안드로이드 앱 개발. <br/>
